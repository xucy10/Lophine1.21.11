- 核心接口层：定义 `ThreadedRegionizer` 和 `ThreadedRegion` 以映射 Folia 的线程化 tick 区域逻辑，支持异步获取区域状态。
- 事件扩展层：在 `me.earthme.luminol.api.entity` 和 `portal` 包下提供针对异步传送（`EntityTeleportAsyncEvent`）和门户交互（`Pre/PostEntityPortalEvent`）的 Bukkit 事件。
- 基础设施层：集成 Pufferfish 的 SIMD 检测与 Sentry 错误监控上下文（`SentryContext`），以及 Leaves 的特性开关管理（`FeatureManager`）。
- 依赖方向：所有类均依赖 Bukkit/Spigot API 及 JetBrains 注解库，作为服务端核心的扩展 API 模块存在。
