- **核心实体层**：定义了 `Bot` 和 `Photographer` 接口，均继承自 Bukkit 的 `Player` 接口，作为虚拟玩家和回放录制者的抽象。
- **动作系统**：采用命令模式设计，通过 `BotAction` 接口及其实现类（如 `MoveAction`, `AttackAction`）定义可执行行为，并由 `BotManager` 统一创建和管理。
- **生命周期管理**：提供 `BotManager` 和 `PhotographerManager` 接口，负责实体的创建、查找、移除以及全局状态维护。
- **事件驱动**：在 `org.leavesmc.leaves.event.bot` 包下定义了完整的 Bot 生命周期事件（如 `BotCreateEvent`, `BotDeathEvent`），支持插件监听与干预。
- **配置与选项**：通过 `BukkitRecorderOption` 等类提供回放录制的细粒度控制（如天气、时间、聊天过滤）。
